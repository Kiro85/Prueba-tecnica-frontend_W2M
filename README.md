# 📑 Technical Test
This project is an application built on a single-page application (SPA) that allows users to manage a superhero roster. The features available on this website include:

- View heroes in a paginated list
- Search by name in the top navigation bar
- Add new heroes
- Edit existing heroes
- Delete existing heroes

> [!IMPORTANT]
> This test covers only the frontend. The **json-server** library was used to simulate the backend.
